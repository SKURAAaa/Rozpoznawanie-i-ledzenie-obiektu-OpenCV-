# Rozpoznawanie-i-ledzenie-obiektu-OpenCV-
Celem było stworzenie aplikacji w Pythonie umożliwiającej:  wykrywanie charakterystycznych punktów obrazu, wyznaczanie deskryptorów cech lokalnych, dopasowanie cech pomiędzy obrazami, lokalizację obiektu na podstawie dopasowań, śledzenie obiektu w sekwencji wideo.  Aplikacja wykorzystuje bibliotekę OpenCV oraz algorytmy detekcji
